# Demo Story

## Positioning Sentence

Actify AI is a secure agentic commerce world where users delegate actions to an AI agent using Auth0-backed permissions and can watch every decision happen in real time.

## What Judges Should Remember

1. This is not just an AI chatbot.
2. This is not just a game skin on top of a dashboard.
3. This is a trust layer for AI action, presented as an explorable world.

## Demo Flow

### 1. Open On Landing Page

Narration:

"Most AI tools tell you what to do. Actify AI lets an agent do it for you, but only within permissions you control."

### 2. Sign In

Narration:

"We start with identity because delegated action without identity is unsafe."

### 3. Enter Dashboard

Narration:

"The dashboard is split into three parts: navigation and controls on the left, the world in the middle, and a live intelligence panel on the right."

### 4. Explore The World

Click a verified shop, a mission hub, and the vault zone.

Narration:

"Each location represents a capability or decision point. The world is how we surface the agent's context."

### 5. Launch A Mission

Mission example:

"Find the best verified item under my budget."

Narration:

"The agent evaluates options inside the user's rules instead of acting with unlimited freedom."

### 6. Show Approval

If above threshold:

"For higher-risk actions, the system requests a stronger confirmation before execution."

### 7. Show Activity Result

Narration:

"Every action gets a reason, a status, and a log entry, so the user can understand what happened and why."

## Demo Checklist

- Landing page loads quickly
- Login works from a clean browser state
- Dashboard opens to a stable default scene
- At least three world entities respond correctly
- One mission completes successfully
- One approval or policy boundary is demonstrated

## Backup Demo Mode

If a live provider has issues:

- keep Auth0 live if possible
- use seeded Convex data
- switch action execution to a safe mocked connector
- keep the exact same UI flow and logging story

## Short Pitch

"We built a secure agentic commerce system where users authorize an AI agent to browse, decide, and act inside a world they can actually understand."
